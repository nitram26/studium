Condicionales IF-SHOW

  1. V-IF

    v-if (Renderizado condicional)

    • Se usa para añadir o quitar elementos del DOM según una condición.
    • Si la condición es false, el elemento no existe en el DOM.
    • Tiene un mayor costo de rendimiento cuando cambia frecuentemente, ya que Vue agrega o elimina el nodo completamente. 

        <template>
      <button @click="mostrar = !mostrar">Mostrar / Ocultar</button>
      <p v-if="mostrar">Este texto aparece y desaparece del DOM</p>
    </template>
    
    <script setup>
    import { ref } from 'vue';
    
    const mostrar = ref(true);
    </script>

    Cuando mostrar es false, desaparece completamente del DOM.

    Regla general:
    • Usa v-if cuando la condición cambia pocas veces.
    • Usa v-show cuando la condición cambia frecuentemente.

  2. V-SHOW

    v-show (Visibilidad condicional)

    • Controla la visibilidad del elemento mediante display: none; en CSS.
    • El elemento siempre está en el DOM, pero se oculta visualmente.
    • Es más eficiente si necesitas mostrar/ocultar elementos con frecuencia, ya que solo cambia un estilo CSS. 

          <template>
        <button @click="visible = !visible">Mostrar / Ocultar</button>
        <p v-show="visible">Este texto solo se oculta visualmente</p>
      </template>
      
      <script setup>
      import { ref } from 'vue';
      
      const visible = ref(true);
      </script>

    Cuando visible es false, sigue en el DOM, pero no es visible.

    Regla general:
    • Usa v-if cuando la condición cambia pocas veces.
    • Usa v-show cuando la condición cambia frecuentemente.
